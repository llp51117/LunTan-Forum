# LunTan-Forum（莫兰迪论坛）

> 大二课程设计小组项目，基于 Servlet + JSP 实现的简易论坛系统

---

## 📝 项目简介
这是一个为大二课程设计打造的小组项目，配合教材内容，涵盖 Servlet、JSP 等核心知识点。
目前为半成品状态，已实现核心业务功能，未来可根据时间进一步迭代优化。

## ✨ 核心功能
- **用户模块**：注册 / 登录 / 个人信息管理
- **内容模块**：发帖 / 回帖 / 点赞 / 收藏
- **社交模块**：好友系统 / 消息通知
- **管理模块**：后台用户管理 / 内容审核

## 🛠️ 技术栈
- **后端**：Java Servlet
- **前端**：JSP + CSS + JavaScript
- **数据库**：MySQL
- **服务器**：Apache Tomcat

## 🚀 如何运行
1. 将项目部署到 Tomcat 服务器
2. 配置 MySQL 数据库连接（修改数据库配置文件）
3. 初始化数据库表结构
4. 启动 Tomcat 服务，访问项目地址
